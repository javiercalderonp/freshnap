# FreshNap Static Interactive Pitch Deck

Sitio 100% estático listo para subir a Vercel.

## Estructura

- `/index.html` menú principal
- `/brands/freshnap/index.html` deck interactivo
- `/brands/freshnap/css/theme.css` tema visual de FreshNap
- `/brands/freshnap/js/main.js` lógica específica del deck
- `/brands/freshnap/assets/` assets locales
- `/shared/css/base.css` estilos compartidos y variables CSS
- `/shared/js/core.js` utilidades compartidas
- `/data/freshnap.json` contenido dinámico desacoplado
- `/vercel.json` rewrites para URL limpia

## Deploy en Vercel

Sube esta carpeta como proyecto estático. No requiere build command.
